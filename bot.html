Customized Answer Bot Chatbot Help

This is a quick exploration of the Zendesk chatbot capabilities and customization options. All the code can be seen in the page source.

The Zendesk widget is located on the bottom right-hand side of the page. Clicking the link will initiate the Answer Bot chatbot. Many elements of the bot are customizable, but it is not yet clear whether the welcome text is one of those those elements.

Searching will return relevant articles from the Zendesk Help Center. Try "credit card" to see some results.

In this version, the "Get in Touch" button is only presented after an initial search is made. This is configurable.
